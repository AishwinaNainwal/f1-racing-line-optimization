# F1 Racing Line Optimization (ML + RL)

This project predicts optimal Formula 1 racing trajectories using real-world telemetry data from FastF1.

## 🚀 Overview
- Supervised Learning: Random Forest + LSTM for trajectory prediction
- Reinforcement Learning: Monte Carlo Policy Gradient agent (REINFORCE)
- Goal: Learn and reproduce optimal racing lines from driver telemetry

## 📊 Results
- R² Score: 0.9999 (Random Forest)
- Mean Track Error: 23.11 units
- RL Lap Completion: 99.17%

## 🛠 Tech Stack
Python, FastF1, Scikit-learn, PyTorch/TensorFlow, OpenAI Gym, Pandas, NumPy, Matplotlib

## 📁 Files
- `Randomforest_Lstm.ipynb` – ML model
- `RL_model.ipynb` – Reinforcement Learning model
